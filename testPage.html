安裝步驟

  1. 安裝ruby
  2. 開 start command prompt with ruby 黑黑的畫面
  3. 下指令安裝 gem install compass
  4. 新增一個資料夾
  5. 下指令指向資料夾
  6. 下指令開一個叫做coffee 的 compass夾~~ compass creat coffee
  7. 知道嗎你已經新增好了 !!! 影片會開始介紹這包新增的是什麼東西...
  8. 學會監控開始寫SCSS (或你要像作者改用Sass)

註解這件事

  • " //註解 " 使用兩條斜線註解，不會被編譯到css檔案中，用在只需要給自己看到的註解
  • " /*註解*/ " 可以利用這樣的註解方式同時在原始檔與css檔中留下紀錄
